From Types to FizzBuzz

28 Mar 2019
  • math
  • software

Introduction

Fizzbuzz is a famous programming interview question whose description is actually fairly straightforward:

For each number from N=1 to N=100, if N is…

  • …disible by 3, output “Fizz”.
  • …divisible by 5, output “Buzz”.
  • …divisible by both 3 and 5, output “FizzBuzz”.
  • …otherwise, output the number itself.

A straighforward problem often leads to a straighforward solution - but of course, we’re not straightforward. That’d be too simple, that’d be â€“ boring.

Baby Steps

We’ll require numbers to do this, but we’re no freeloaders. Over here we serve our own hand-crafted artisinal numbers.

struct Zero; // ahem, zero.
struct S<N>; // AKA N + 1

Of course things aren’t as easy as they first appear, but rustc is quite helpful

error[E0392]: parameter `N` is never used
 --> src/lib.rs:2:10
  |
2 | struct S<N>;
  |          ^ unused type parameter
  |
  = help: consider removing `N` or using a marker such as `std::marker::PhantomData`

Let’s go ahead and do just that.

use std::marker::PhantomData;

struct Zero; // ahem, zero.
struct S<N>(PhantomData<N>); // AKA N + 1

Add a few helpful aliases…

type N1 = S<Zero>;
type N2 = S<N1>;
type N3 = S<N2>;
type N4 = S<N3>;
type N5 = S<N4>;

From a walk to a sprint

It’s a long ways to 100 down this route, so let’s forge another path.

trait AddImpl<U> { type Sum; }

// N + 0 = N
impl<T> AddImpl<Zero> for S<T> {
    type Sum = S<T>;
}

// 0 + N = N
impl<T> AddImpl<T> for Zero {
    type Sum = T;
}

// N + (M + 1) = (N + M) + 1
impl<T, U> AddImpl<S<U>> for S<T>
    where
        T: AddImpl<S<U>>
{
    type Sum = S<T::Sum>;
}

type Add<LHS, RHS> = <LHS as AddImpl<RHS>>::Sum;

We can do better.

trait AddImpl<U> { type Sum; }

// N + 0 = N
impl<T> AddImpl<Zero> for S<T> {
    type Sum = S<T>;
}

// 0 + N = N
impl<T> AddImpl<T> for Zero {
    type Sum = T;
}

// N + (M + 1) = (N + M) + 1
impl<T, U> AddImpl<S<U>> for S<T>
    where
        T: AddImpl<S<U>>
{
    type Sum = S<T::Sum>;
}

type Add<LHS, RHS> = <LHS as AddImpl<RHS>>::Sum;
